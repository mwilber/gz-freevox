# FreeVox

Real-time, two-way LLM conversation using WebSockets and OpenAI's API.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Configure environment:

```bash
cp .env.example .env
```

Add your OpenAI API key to `.env`.

3. Run the server:

```bash
npm start
```

Visit `http://localhost:3000`.
